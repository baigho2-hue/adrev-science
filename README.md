## 🌟 Pourquoi AdRev ?

Marre des protocoles rejetés pour "manque de rigueur" ou des erreurs d'échantillonnage qui ruinent des mois de travail ? AdRev est votre bouclier contre les rejets académiques.

### 🧩 Fonctionnalités Clés
- **Assistant de Rédaction IMRAD/SPIRIT** : Validation en temps réel selon la Taxonomie de Bloom.
- **Moteur Statistique Avancé** : 13 types d'échantillonnage avec calcul automatique de l'effet de plan (Deff).
- **Suggesteur de Tests Statistiques** : Recommandations intelligentes (ANOVA, Chi2, Student) basées sur vos variables.
- **Atelier Qualitatif & Codage Thématique** : Pour une analyse rigoureuse des verbatims.
- **Exportation Instantanée (.docx)** : Générez vos manuscrits prêts à l'envoi en un clic.

## 🛠️ Stack Technique
- **Front-end** : Windows Presentation Foundation (WPF) avec design Premium/Glassmorphism.
- **Back-end** : C# .NET 8 (Core Logic & Data Mining).
- **Mathématiques** : MathNet.Numerics (Calculs Inférentiels).
- **Sécurité** : Système de licence cryptographique par Identifiant Matériel (HWID).

## 🚀 Démarrage Rapide

1. **Installation** : Téléchargez le dernier installeur (Release GitHub).
